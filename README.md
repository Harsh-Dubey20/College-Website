College Website Landing Page
A clean, modern, and responsive landing page for a college website. This project focuses on a sleek user interface, seamless navigation, and mobile responsiveness using pure HTML, CSS, and vanilla JavaScript.

Features
1.Responsive Design: Fully optimized for desktops, tablets, and mobile devices using CSS Media Queries.

2.Toggle Mobile Menu: A smooth, slide-in navigation menu for smaller screens powered by JavaScript.

3.Typography & Icons: Integrated with Google Fonts for clean typography and Font Awesome for modern vector icons.

How to Run the Project Locally
1.Clone or download this repository to your local machine.

2.Open the project folder in Visual Studio Code.

3.Open index.html directly in any modern web browser, or use the Live Server extension in VS Code for a real-time preview.
