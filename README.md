# College Website Landing Page

A clean, modern, and responsive landing page for a college website. This project focuses on a sleek user interface, seamless navigation, and mobile responsiveness using pure HTML, CSS, and vanilla JavaScript.

---

## Features

* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices using explicit CSS Media Queries.
* **Toggle Mobile Menu:** A smooth, slide-in sidebar navigation menu for smaller screens, powered by lightweight vanilla JavaScript.
* **Modern Typography & Icons:** Integrated with Google Fonts for clean, professional text rendering and Font Awesome for sharp vector icons.

---

## Tech Stack & Tools

| Component | Technology / Tool Used |
| :--- | :--- |
| **Structure** | HTML5 |
| **Styling** | CSS3 (Flexbox, Media Queries) |
| **Interactivity** | Vanilla JavaScript |
| **Fonts** | Google Fonts |
| **Icons** | Font Awesome |
| **Editor** | Visual Studio Code |
